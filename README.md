# Hướng dẫn chạy dự án

### B1: Lấy code từ git về

```bash
git clone https://github.com/TranHai15/facebook.git
```

### B2: Di chuyển vào dự án

```bash
cd facebook
```

### B3: Di chuyển vào thư mục Frontend và Backend tải tài nguyên và chạy dự án

```bash
cd Backend
```

```bash
npm install
```

```bash
npm run dev
```

```bash
cd Frontend
```

```bash
npm install
```

```bash
npm run dev
```
